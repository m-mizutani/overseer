Overseer
===============


Setup
--------------

```sh
$ mkdir -p $GOPATH/src/github.com/m-mizutani
$ cd $GOPATH/src/github.com/m-mizutani
$ git clone https://github.com/m-mizutani/overseer.git
$ cd overseer
$ dep ensure
$ go build
```
